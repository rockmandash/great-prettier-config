# `@rockmandash/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ yarn add --dev @rockmandash/prettier-config
```

```bash
$ npm install -D @rockmandash/prettier-config
```

**Edit `package.json`**:

```json
{
  // ...
  "prettier": "@rockmandash/prettier-config"
}
```
